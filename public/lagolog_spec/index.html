    Note: Command that begin with “WITH” will be interpreted in parallel with the next dialogue message.

    WITH SOUND: "SOUND_ID";

    Custom Callbacks #

    Note: Command that begin with “WITH” will be interpreted in parallel with the next dialogue message.

    WITH CALLBACK: "CALLBACK_ID";

    Sprite Management #

    WITH CG: "CG_ID";

    Hide CG # 

    WITH EXIT CG;
